## 项目概述

SmartVoice 是一款智能多角色有声内容创作 Web 应用，旨在将小说、剧本、散文等文本内容转化为专业级多角色有声作品。

**核心价值主张：** 让每个人都能轻松创作专业级的多角色有声内容，让文字生动起来。

**目标用户：**
- 内容创作者（20%）：网文作家、有声书制作人
- 文学爱好者（60%）：想自制有声内容的普通用户
- 学习者（20%）：通过听学习的用户

## 项目状态

当前处于 **产品规划阶段**（2026-01-20），尚未开始代码开发。

## 技术架构（规划中）

根据产品规划文档，技术栈计划如下：

- **前端：** React + TypeScript + Tailwind CSS
- **后端：** Node.js / Python
- **TTS：** 讯飞/百度 API（优先讯飞）
- **NLP：** BERT-based 角色识别模型
- **存储：** 阿里云 OSS
- **数据库：** MySQL / PostgreSQL

## 核心功能（MVP）

1. **文本上传与格式解析** - 支持 txt、epub 格式
2. **智能角色识别与自动分配** - AI 自动识别角色、对话、旁白
3. **音色预览和高级定制** - 20+ 种音色库，支持音速、音调、语调、情绪调节
4. **批量章节生成** - 支持多章节批量生成，保持角色音色一致性
5. **情绪和语调智能理解** - 基于文本内容自动识别情绪
6. **历史记录和角色设置复用** - 保存项目配置
7. **音频预览和导出** - 在线预览播放，下载 MP3 格式
